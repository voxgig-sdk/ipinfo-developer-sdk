# ipinfo-developer-mcp

MCP server exposing the IpinfoDeveloper SDK as tools, built on the
[official Go MCP SDK](https://github.com/modelcontextprotocol/go-sdk)
and the sibling Go SDK at `../go`.

## Tools

| Tool | Args | Returns |
|------|------|---------|
| `ipinfo-developer_list` | `entity`, optional `query` map | First page of records as JSON |
| `ipinfo-developer_load` | `entity`, `query` (e.g. `{id:N}`) | Single record as JSON |

JSON schemas for both tools are emitted by the SDK from the `Args`
struct's `json` / `jsonschema` tags — no schema is hand-written.

## Entities

abuse | asn | carrier | company | core | domain | general | get_current_information | get_information_by_ip | ipinfo_core | ipinfo_lite | ipinfo_plus | lite | max | men | place | plus | privacy | privacy_extended | range | residential_proxy | single | whois_asn | whois_domain | whois_ip | whois_net_id | whois_org | whois_poc

## Build

```sh
go build -o ipinfo-developer-mcp ./...
```

## Run

```sh
# stdio transport — for spawnable agent installs (default).
./ipinfo-developer-mcp

# streamable HTTP transport — share one running server between agents.
./ipinfo-developer-mcp -transport http -addr :8080
```

## Install for Claude Code

```sh
claude mcp add --scope user ipinfo-developer \
  -- /absolute/path/to/ipinfo-developer-mcp -transport stdio
```

After install, restart Claude Code; the `ipinfo-developer_list` and
`ipinfo-developer_load` tools become available in new sessions.

## Smoke test via HTTP

```sh
./ipinfo-developer-mcp -transport http -addr :18080 &

# initialize, grab the session id
curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -D headers \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-06-18","capabilities":{},"clientInfo":{"name":"smoke","version":"0"}}}'

SESSION=$(awk '/Mcp-Session-Id/ {print $2}' headers | tr -d '\r')

curl -sN -X POST http://localhost:18080 \
  -H 'Content-Type: application/json' \
  -H 'Accept: application/json, text/event-stream' \
  -H "Mcp-Session-Id: $SESSION" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ipinfo-developer_load","arguments":{"entity":"abuse","query":{"id":1}}}}'
```
